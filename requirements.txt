# ============================================================
# PyTorch — instalar POR SEPARADO según plataforma. Sin fijar versión
# exacta: los pins antiguos (p. ej. 2.7.1+cpu) dejan de estar disponibles
# en el índice de PyTorch con el tiempo; dejamos que pip resuelva la más
# reciente compatible entre torch y torchaudio.
#
# Windows/Linux CPU (sin GPU):
#   pip install torch torchaudio --index-url https://download.pytorch.org/whl/cpu
#
# Linux con CUDA 12.x:
#   pip install torch torchaudio --index-url https://download.pytorch.org/whl/cu121
#
# Raspberry Pi (ARM, sin GPU):
#   pip install torch torchaudio
#   (usa las wheels oficiales ARM de PyPI — más lentas)
# ============================================================

# ============================================================
# TTS — instalar POR SEPARADO (no en PyPI estándar):
#
# Python 3.12 (requiere fork de Idiap):
#   pip install git+https://github.com/idiap/coqui-ai-TTS
#
# Python ≤ 3.11:
#   pip install TTS
# ============================================================
